# Patlast
## _The last atlas packer youll need, Ever..._

Patlast is a free and open-source texture atlas packer designed for:

- Voxel engines
- Art programs
- Game / Rendering engines
- Artists
- Anybody else!

## Features

- Supports multiple grouping and packing styles.
- Export respecting size constraints.

Patlast also supports diffrent ways of managing your atlas, such as:

> "Single"
> Images are grouped into a single image and organized by size.

> "Grouped"
> Images are grouped into the smallest size catagory they fit in, if a catagory could not be found they are put into a new one. Each catagory is double the size of the last.
